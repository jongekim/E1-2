# 🎯 나만의 퀴즈 게임

## 프로젝트 개요

이 프로젝트는 Python으로 개발된 터미널 기반 퀴즈 게임입니다. 사용자는 다양한 주제의 퀴즈를 풀고, 새로운 퀴즈를 추가하고, 점수를 기록할 수 있습니다.

프로그램을 종료해도 모든 데이터가 `state.json` 파일에 저장되어 유지됩니다.

## 퀴즈 주제 선정 이유

**IT/프로그래밍 관련 퀴즈**를 선택했습니다. 이유는:
1. 현재 학습 중인 분야라 흥미로움
2. Python, JavaScript, Git, Docker 등 다양한 IT 개념을 포함 가능
3. 실무에서 필요한 지식을 체계적으로 정리하는 데 도움
4. 정기적으로 퀴즈를 추가하여 지속적인 학습 가능

## 실행 방법

### 요구 사항
- Python 3.10 이상
- 외부 라이브러리 없음 (표준 라이브러리만 사용)

### 실행 명령어
```bash
python main.py
```

### 첫 실행
프로그램을 처음 실행하면 기본 10개의 IT/프로그래밍 퀴즈가 자동으로 로드됩니다.

## 기능 목록

### 1. 🎮 퀴즈 풀기
- 저장된 모든 퀴즈를 순서대로 풀이
- 각 문제마다 4개의 선택지 제시
- 정답/오답 즉시 확인
- 전체 문제 풀이 후 최종 점수 표시

### 2. ➕ 퀴즈 추가
- 새로운 퀴즈를 자유롭게 등록
- 문제, 4개의 선택지, 정답 번호 입력
- 추가된 퀴즈는 즉시 저장됨

### 3. 📋 퀴즈 목록
- 등록된 모든 퀴즈의 문제만 표시
- 퀴즈 개수 확인 가능

### 4. 🏆 점수 확인
- 지금까지의 최고 점수 확인
- 정답한 문제수 및 백분율 표시

### 5. 💾 자동 저장
- 모든 데이터가 `state.json`에 자동 저장
- 프로그램 재시작 후에도 데이터 유지

## 데이터 파일 설명 (state.json)

### 파일 경로
프로젝트 루트 디렉터리: `./state.json`

### 파일 역할
- 사용자가 추가한 모든 퀴즈를 JSON 형식으로 저장
- 최고 점수를 기록
- 프로그램 재실행 시 자동으로 로드

### 인코딩
UTF-8 (한글 지원)

### 스키마 (데이터 구조)

```json
{
    "quizzes": [
        {
            "question": "퀴즈 문제 텍스트",
            "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
            "answer": 1
        }
    ],
    "best_score": 10
}
```

**필드 설명:**
- `quizzes` (배열): 등록된 모든 퀴즈
  - `question` (문자열): 퀴즈 문제
  - `choices` (배열): 4개의 선택지
  - `answer` (정수): 정답 번호 (1-4)
- `best_score` (정수): 최고 점수 (정답한 문제 수)

### 데이터 손상 시 복구
파일이 손상되거나 읽을 수 없으면 자동으로 기본 데이터로 복구됩니다.

## 코드 구조

### 클래스 설계

#### Quiz 클래스 (`quiz.py`)
개별 퀴즈를 표현하는 클래스
- **속성**: question, choices, answer, hint
- **메서드**: 
  - `display()`: 퀴즈 화면에 출력
  - `check_answer()`: 정답 확인
  - `show_hint()`: 힌트 표시 (보너스)
  - `to_dict()`: JSON 직렬화
  - `from_dict()`: JSON 역직렬화

#### QuizGame 클래스 (`quiz_game.py`)
게임 전체를 관리하는 클래스
- **속성**: quizzes, best_score
- **메서드**:
  - `run()`: 메인 루프
  - `display_menu()`: 메뉴 출력
  - `play_quiz()`: 퀴즈 풀기
  - `add_quiz()`: 퀴즈 추가
  - `display_quiz_list()`: 목록 표시
  - `display_best_score()`: 점수 표시
  - `save_data()`: 파일에 저장
  - `load_data()`: 파일에서 로드

#### 유틸리티 함수 (`utils.py`)
입력 검증 관련 함수
- `get_valid_integer()`: 범위 내의 정수 입력받기
- `get_non_empty_string()`: 공백이 아닌 문자열 입력받기

## 예외 처리

프로그램은 다음 상황을 안전하게 처리합니다:
- **숫자 변환 실패**: "abc" 같은 잘못된 입력
- **범위 초과**: 허용 범위 밖의 숫자 입력
- **빈 입력**: Enter만 눌렀을 경우
- **Ctrl+C**: KeyboardInterrupt 발생 시
- **입력 스트림 종료**: EOFError 발생 시
- **파일 손상**: state.json이 손상되었을 경우
- **파일 부재**: 첫 실행 시 기본 데이터 사용
