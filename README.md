# 🎯 나만의 퀴즈 게임

## 프로젝트 개요

이 프로젝트는 Python으로 개발된 터미널 기반 퀴즈 게임입니다. 사용자는 퀴즈를 풀고, 새로운 퀴즈를 추가하고, 점수를 기록할 수 있습니다.

```bash
python main.py
```

## 기능 목록

- 🎮 **퀴즈 풀기**: 저장된 퀴즈를 풀고 점수를 획득합니다
- ➕ **퀴즈 추가**: 새로운 퀴즈를 등록합니다
- 📋 **퀴즈 목록**: 등록된 모든 퀴즈를 확인합니다
- 🏆 **점수 확인**: 최고 점수를 확인합니다
- 💾 **자동 저장**: 모든 데이터는 state.json에 자동으로 저장됩니다

## 데이터 파일 설명 (state.json)

### 경로
프로젝트 루트 디렉터리: `./state.json`

### 역할
- 사용자가 추가한 모든 퀴즈를 저장합니다
- 최고 점수를 기록합니다
- 프로그램 재실행 시에도 데이터가 유지됩니다

### 필드 구조
```json
{
    "quizzes": [
        {
            "question": "문제",
            "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
            "answer": 1
        }
    ],
    "best_score": 0
}
```

- `quizzes`: Quiz 객체 리스트
  - `question`: 문제 텍스트
  - `choices`: 4개의 선택지 배열
  - `answer`: 정답 번호 (1-4)
- `best_score`: 최고 점수 (퀴즈를 푼 정답 개수)
